ADDING NEW SKILLS
─────────────────────────────────────────────────────────────────────────────

1. Create YAML file:
   cat > config/agent-skills/my-skill.yaml << 'YAML'
   skill:
     name: "my-skill"
     category: "rdf"  # or other category
     version: "1.0.0"
     agent_types: ["agent-type"]
     description: "..."
     capabilities: [...]
     performance_slo:
       max_duration_ms: 5000
       success_rate: 0.99
     # ... other fields
   YAML

2. Validate:
   skill_validate_file config/agent-skills/my-skill.yaml

3. Load and register:
   skill_load "my-skill" | skill_register

TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

Skill not found:
  • Verify file exists: ls config/agent-skills/skill-name.yaml
  • Validate file: skill_validate_file config/agent-skills/skill-name.yaml
  • Register: skill_load "skill-name" | skill_register

Validation fails:
  • Check required fields present
  • Verify category is valid
  • Ensure SLO values: max_duration_ms >= 100, success_rate 0.80-1.0

Registry corrupted:
  • Clear: rm config/agent-skills/.registry.json
  • Rebuild: for f in config/agent-skills/*.yaml; do
      skill_load $(basename $f .yaml) | skill_register
    done
