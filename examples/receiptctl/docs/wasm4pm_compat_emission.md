# wasm4pm-compat emission surface

Generated by `wasm4pm-compat-pack`. This project **emits** OCEL 2.0 events
(`wasm4pm_compat::ocel::OCELEvent`); it never analyses them. Discovery,
conformance, fitness, precision, and variant extraction are owned by
`wasm4pm-compat` / `wasm4pm`.

## Emitted event types

| `graph_union_hashed` | `OntologyGraph` | `emit_graph_union_hashed` |
| `pack_lock_verified` | `Pack` | `emit_pack_lock_verified` |
| `receipt_chained` | `SyncReceipt` | `emit_receipt_chained` |

See `src/wasm4pm_compat_events.rs` for the generated `EmittedEventType` enum.
