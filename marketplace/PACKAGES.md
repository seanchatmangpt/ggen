## üöÄ Getting Started

1. **Search** for packages matching your needs:
   ```bash
   ggen market search "your query"
   ```

2. **Get Details** about a package:
   ```bash
   ggen market info "package-name"
   ```

3. **Install** the package:
   ```bash
   ggen market install "package-name"
   ```

4. **Read the README** in the installed package for usage instructions

5. **Explore Examples** in the package's examples directory
