# W1: Full Worktree Inventory

All files (sources, tests, examples, queries, documentation, and scripts) are fully cataloged with size, modification times, and cryptographic hashes in `worktree_inventory.full.json`.
This prevents shadow commits or untracked changes during agent lifecycle execution.
