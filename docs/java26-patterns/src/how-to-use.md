## The Executable Pattern UX

Every pattern follows the same three-step invocation:

1. **Define your domain in TTL** — `.specify/domain.ttl`
2. **Configure ggen.toml** — add a `[[generation.rules]]` entry
3. **Generate** — run `ggen sync`
