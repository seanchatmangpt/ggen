## Security Update SLAs

- **Critical (CVSS 9.0-10.0)**: Patch within 24 hours
- **High (CVSS 7.0-8.9)**: Patch within 7 days
- **Medium (CVSS 4.0-6.9)**: Patch within 30 days
- **Low (CVSS 0.1-3.9)**: Patch in next scheduled release

## Acknowledgments

We appreciate responsible disclosure and will acknowledge security researchers in our release notes (with permission).

This project implements continuous security scanning:

- Secret detection (TruffleHog)
- License compliance checking

## Responsible Disclosure Guidelines

We ask that you:

1. **Do not** disclose vulnerabilities publicly until a patch is available
2. **Do** give us 30 days to release a patch before public disclosure
3. **Do** test patches before reporting that they're ineffective
4. **Do** allow us to credit you in security advisories (optional)

## Bug Bounty

Currently, we do not offer a formal bug bounty program. However:
- Security researchers who report valid vulnerabilities will be acknowledged
- Significant findings may be considered for recognition/rewards
- Contact security@tai-autonomics.dev for inquiries
