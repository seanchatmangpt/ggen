### Cash Preservation Levers (If Needed)

If cash runway shortens unexpectedly:

1. **Hiring pause** (30-day reduction in burn)
   - Only backfills critical roles
   - Contractors paused (save $50K/month)
   - Impact: 3-month runway extension

2. **Salary reduction** (20% company-wide, temp)
   - CEO leads with 25% cut
   - All team 20% (temporary, revert with profitability or Series A)
   - Impact: 2-month runway extension
   - Legal: Document temporary reduction + make-good commitment

3. **Marketing/Travel freeze** (save $8K/month)
   - No conferences/events (pause $5K/month)
   - No business travel except customer-critical (save $3K/month)
   - Impact: 1-month runway extension

4. **Infrastructure optimization** (save $3-5K/month)
   - Migrate from paid to open-source tools
   - Consolidate cloud services
   - Impact: 1-month runway extension

**Combined levers if needed**:
- Hiring pause + salary reduction + marketing freeze = 6-month runway extension
- Only triggered if Series A delayed >2 months past target

## Appendix: Burn Calculation Methodology

**Total Monthly Burn Formula**:

```
Burn = Payroll + Benefits + Infrastructure + Sales/Marketing + Legal/Compliance + G&A - Revenue

Payroll: All employee salaries (prorated if mid-month hire)
Benefits: 15-18% of payroll (health, 401k, taxes, workers comp)
Infrastructure: GCP, Pub/Sub, monitoring ($5K base + 20% for growth)
Sales/Marketing: $2-5K pre-Series A, $30K+ post-Series A
Legal/Compliance: $2K/month base + $5-15K for milestones (409A, SOC2 init)
G&A: $2-5K for office, tools, misc
Revenue: MRR Ã— 12 (annualized for monthly view)
```

**Example (Month 7)**:
```
Payroll: $1.312M
Benefits (17%): $223K
Infrastructure: $8K
Sales/Marketing: $8K
Legal/Compliance: $5K
G&A: $5K
Revenue: -$30K (MRR of $30K annualized)

Total Burn = $1.312M + $223K + $8K + $8K + $5K + $5K - $30K = $1.531M
```
