=============================================================================
USAGE IN GGEN
=============================================================================

These ontologies form the semantic foundation for the ggen code generation
system and are loaded during the μ₁ (load) stage of the five-stage pipeline:

  μ₁ (load)          → Loads these core ontologies + project-specific specs
  μ₂ (extract)       → Extracts patterns from merged RDF graph
  μ₃ (generate)      → Generates code using Tera templates
  μ₄ (validate)      → Validates output against SHACL shapes
  μ₅ (emit)          → Writes final artifacts

Configuration in ggen.toml:
  [ontologies]
  core_path = "ontologies/core/"
  load_rdfs = true
  load_owl = true
  load_skos = true
  load_dcat = true
