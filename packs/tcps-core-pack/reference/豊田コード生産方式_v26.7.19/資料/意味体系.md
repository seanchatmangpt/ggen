# 意味体系

## 原初の二本柱

```text
自働化
├── 異常がわかる
├── 異常で止まる
├── 異常で止める
├── 不良を後工程へ流さない
└── 人を機械の番人にしない

必要時生産
├── 必要なもの
├── 必要な時
├── 必要なだけ
├── 後工程が引き取る
└── 引取量だけ補充する
```

## 系譜保存

```text
原問題 → 発明 → 不変条件
```

## 自働化状態遷移

```text
稼働中 → 異常停止 → 停止中 → 真因対策 → 対策中 → 標準更新 → 再開可能 → 稼働中
```

## Auto Select

```text
意味 → 適格性 → 準備完了 → 質量 → 選択提案 → 仲介者許可 → 実行
```
